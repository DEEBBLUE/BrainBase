# Опредление
Будем называть пространством исходом(или элементарных событий) конечное [[Множество|множетсво]] всевозможных исходом эксперамента A и будем его обозначать буквой $\Omega$ и запсывать в виде :$$\Omega=\{\omega_1,\omega_2,...,\omega_n\}$$
Сами исходы $\omega_1,\omega_2,...,\omega_n$ называются элементарными событиями
# Примеры
## Пример 1
Запишем пространство исходов для эксперемента в котором мы n-раз подбрасываем монетку.Тогда пространство исходов будет слудующим :$$\Omega=\{\omega: \ \omega=(a_1,a_2,...,a_n) \ ,\ Где \ \ a_i = О \ \ или \ \ Р\}$$
[[Множество#Мощность множетсва|мощность]] же такого множетсва очевидно равна $2^n$
