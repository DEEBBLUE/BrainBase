Пусть задано [[Пространстово исходов|пространство исходов]] $\Omega$ и какое-то его [[Разбиение|разбиение]] $\mathcal{D}$ 
Докажем,что множество $\mathcal{A}$ все возможных объединений [[Разбиение|атомов]] $\mathcal{D}$ $\mathcal{B} = \{ B: \ B_i \ = \ D_{i_1} \cup D_{i_2} ... \ \cup D_{i_m} \}$ и пустое множетсво $\emptyset$  создают алгебру.То есть выполнуются следующие утверждние:
1)$\Omega \in \mathcal{A}$
2)$\forall A,B \in \mathcal{A}$ выполняется следующее $A\cap B,A\cup B,A /B \in \mathcal{A}$ 
Проверим первое утверждение относительно $\mathcal{A}$ .По опредлению $\mathcal{D}$ объединение всех атомов этого разбиение образуют $\Omega$ ,а уже по определению $\mathcal{A}$ в ней будет содержаться объединение всех атомов $\mathcal{D}$ ,а значит $\Omega \in \mathcal{A}$ 
Теперь проверим второе утверждение.Пусть у нас есть $A,B \in \mathcal{A}$.
По определению $\mathcal{A}$ A и B можно записать следующим образом:$$A=D_{i_1}\cup D_{i_2} ... \cup \ D_{i_q}$$$$B=D_{j_1}\cup D_{j_2} ... \cup \  D_{j_p}$$
Теперт докажем сначала,что $A\cup B\in\mathcal{A}$.Это верно тк объедение этих множеств по сути тоже есть объедение атомов из разбиения $\mathcal{D}$ .
Теперь докажем,что $A\cap B\in\mathcal{A}$.Пусть множества A и B состоят из объедения различных атомов разбиения,тогда очевидно,что их пересечение есть пустое множество,а оно принадлежит $\mathcal{A}$.Пусть всё таки в A и B есть одинаковые атомы из разбиения,тогда их пересечение тоже будет какм-то объединеним этих атомов,и оно тоже лежит в $\mathcal{A}$
Факт того ,что $A|B\in\mathcal{A}$ я доказывать не буду,потому что он слишком тривиален
Из выше скозанного [[Алгебра пораждённая разбиением D|алгеброй пораждённой разбиеним D]] называется объединение множетсва всевозможных объединений атомов этого разбиения и пустого множетсова,обозначается эта алгебра как $\alpha(\mathcal{D})$
