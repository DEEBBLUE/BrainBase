# Опредления 
Существует несколько опредлений,что такое предел последовательности,я их приведу и докажу их эквивелентность 
## Определение Коши(на $\varepsilon -\delta$ языке)
Пределом последовательности X называют такое число А,что 
$$(\lim_{n\rightarrow\infty}x_n=A)\iff(\forall \varepsilon > 0 \ \exists N\in\mathbb{N}:\forall n\in\mathbb{N} \ \   n> N : |x_n-A|<\varepsilon)$$
## Опредление через окрестности
Пределом последовательности X называют такое число А,что $$(\lim_{n\rightarrow\infty}x_n=A)\iff(\forall V(A) \ \ \exists N\in\mathbb{N} \ \ \forall n > N (x_n\in V(A)))$$
## Докозательство эквивалентности [[Матан/МатАнализ/Предел/Предел последовательности/Предел последовательности#Определение Коши(на $ varepsilon - delta$ языке)|определения 1]] и [[Матан/МатАнализ/Предел/Предел последовательности/Предел последовательности#Опредление через окрестности|определения 2]] 
Сначала докажем,что из 1-го следует 2-е.То есть у нас есть последовательность Х,предел которой $$(\lim_{n\rightarrow\infty}x_n=A)\iff(\forall \varepsilon > 0 \ \exists N\in\mathbb{N}:\forall n\in\mathbb{N} \ \   n> N : |x_n-A|<\varepsilon)$$
Условием $\forall\varepsilon >0$ и $|x_n-A|<\varepsilon$ мы по сути задаём произвольную эпсилон окрестность точки А,а находя расстояние между A и $x_n$ мы пытаемся определить,лежит ли $x_n$ внутри этой $V(A)$.То есть всё выше сказанное можно записать в виде:$$(\lim_{n\rightarrow\infty}x_n=A)\iff(\forall V(A) \ \ \exists N\in \mathbb{N} \ \ \forall n >N \ \ x_n\in V(A))$$
Теперь покажем,что из 2-го следует 1-ое.Значит у нас есть последовательность Х и $$(\lim_{n\to\infty}x_n=A)\iff(\forall V(A) \ \ \exists N\in\mathbb{N} \ \ \forall n>N \ \ x_n\in V(A))$$
По опредлению $V(A)=\{x: A-\varepsilon < x < A+\varepsilon\} \Rightarrow \{x:|A-x|<\varepsilon\}$
