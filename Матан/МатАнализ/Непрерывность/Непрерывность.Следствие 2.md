Если $f:E\rightarrow\mathbb{R}$ и она непрерывна в точке $a\in E$ ,которая является [[Предельная точка|предельной точкой]] ,тогда $$(f:E\rightarrow\mathbb{R}  \ \ \ непрерывна \ \ в \ \ точке \ \ а \in E \ и \ a \ предельная \ точка \ для \ E)\iff (\lim_{E\ni x \rightarrow a} f(x)=f(a))$$ 
Доказательство:
Пусть есть $f:E\rightarrow\mathbb{R}$ которая непрерывна в точке $a\in E$,тогда по опредлению [[Непрерывность.Определения#В виде окрестностей|непрерывной функции в точке]] для любой окрестности $V_\mathbb{R}^\epsilon(f(a))$ существует $\mathring{U}_E^\delta(a)$ ,такая ,что $f(\mathring{U}_E^\delta(a))\subset V_\mathbb{R}^\epsilon(f(a))$ ,это утверждение в точности повторяет [[Предел фукции|определение предела функции]].
Теперь пусть существует предел $$\lim_{E\ni x \rightarrow a}f(x)=f(a)$$
Тупо очевидно крч
Из этого соотношения можно получить,что 
$$\lim_{E\ni x \rightarrow a}f(x) = f(\lim_{E\ni x \rightarrow a}x)$$
