# Определение
## На $\epsilon-\delta$  языке:
[[Отображение|Функция]] $f:E\rightarrow\mathbb{R}$ определённая на [[Множество|множетстве]] $E$ называется непрерывной в точке $a\in E$,если для любой окрстности $V(f(a))$ существует окрестность [[Окрестность произвольного множества,являющегося подмножеством R|$U_E(a)$]] ,такая ,что $f(U_E(a))\subset$ $V(f(a))$ 
Ну или тоже самое но на $\epsilon$-$\delta$ языке:
$$(f:E \rightarrow \mathbb{R} \ \ непрерывна в точке \ \ a) \iff (\forall \epsilon >0 \ \ \exists\delta>0 \ \ \forall x\in E :|x-a|<\delta ,\ \ то \ \ |f(x)-f(a)| < \epsilon ))$$
### В виде окрестностей:
$$(f:E \rightarrow \mathbb{R} \ \ непрерывна в точке \ \ a) \iff(\forall V_\mathbb{R}^\epsilon(f(a)) \ \   \exists U_E^\delta(a) \ \\ \ (f(U_E^\delta(a)) \subset V_\mathbb{R}^\epsilon(f(a))) $$
# Следствия 
## Первое следствие
Если a - [[Предельная точка#Изолированная точка| изолированная]] точка,тогда найдётся $U(a)$ в кторой нет других элементов кроме а.Тогда $U_{E}(a)=a$ и $f(U_{E}(a))=f(a)\subset V(f(a))$ 
## Второе следствие 
Если $f:E\rightarrow\mathbb{R}$ и она [[Непрерывность|непрерывна]] в точке $a\in E$ ,которая является [[Предельная точка|предельной точкой]] ,тогда $$(f:E\rightarrow\mathbb{R}  \ \ \ непрерывна \ \ в \ \ точке \ \ а \in E \ и \ a \ предельная \ точка \ для \ E)\iff (\lim_{E\ni x \rightarrow a} f(x)=f(a))$$ 
Доказательство:
Пусть есть $f:E\rightarrow\mathbb{R}$ которая непрерывна в точке $a\in E$,тогда по опредлению [[Непрерывность.Определения#В виде окрестностей|]] [[Непрерывность|непрерывной функции в точке]]  для любой окрестности $V_\mathbb{R}^\epsilon(f(a))$ существует $\mathring{U}_E^\delta(a)$ ,такая ,что $f(\mathring{U}_E^\delta(a))\subset V_\mathbb{R}^\epsilon(f(a))$ ,это утверждение в точности повторяет [[Предел фукции|определение предела функции]].
Теперь пусть существует предел $$\lim_{E\ni x \rightarrow a}f(x)=f(a)$$

Из этого соотношения можно получить,что 
$$\lim_{E\ni x \rightarrow a}f(x) = f(\lim_{E\ni x \rightarrow a}x)$$
