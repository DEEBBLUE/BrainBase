# Определения
## Определение 1
Точка $a\in E$ называется предельной точкой множетсва E если,в любой окрестности [[Выколотая окрестность|выколотой]] окрестности $\mathring{U}(a)$ бесконечное количество элементов множества $E$ 
## Определение 2
Так же эквивалентной этой формулировке является формулировка,что в любой [[Выколотая окрестность|выколотой]] окрестности $\mathring{U}(a)$ найдётся хотя бы 1 элемент из $E$

## Доказательство эквивалентности [[Предельная точка#Определение 1|опредления один]] и [[Предельная точка#Определение 2|определения два]]
Докажем сначало,что первое эквивалентно второму,по первому определению в окрестности $\mathring{U}(a)$ лежит бесконечное количество элементов множества $E$ ,то есть это будет удовлетворять второму определению,тк по нему нужно ,чтобы был хотя бы 1,очевидно,что в бесконечном числе элментов есть этот минимально необходимый 1 элемент.
Тебе докажем,что из второго опредления следует первое.Исходя из 2-го опредления в любой окрестности $\mathring{U^\varepsilon}(a)$ лежит хотя бы 1 элемент из $E$,то есть есть такой $q_1\in E:a-\varepsilon < q_1 < a+\varepsilon$  ,теперь рассмотрим окрестноть $\mathring(U^{q_1})(a)$ внутри неё будет точка $q_2\in E:q_1-a<q_2<q_1+a$ потому что в [[Предельная точка#Определение 2|определении]] сказано,что в любой окрестности,есть хотя бы одна,мы можем проделать так бесконечное количество раз и получить множество $Q=\{q_1,q_2,...,q_n,...\}$ $Q \subset E$ .Все элементы множества Q лежат внутри $\mathring{U^\varepsilon}(a)$ ,то есть внутри этой окрестнти лежит бескнонечное количество элементов из E,что соотвествует первому определению.
