# Опредления 
## Бинарные отношения
Бинарным отношением X и Y называют $\omega\subset X\times Y$.Для упорядоченной пары $(x,y)\in\omega$ пишут $x\omega y$ 
## Отношение эквивалентости 
Отношением эквивалентоности называют [[Отношения#Опредления#Бинарные отношения|бинарное отношение ~]] такое ,что для $\forall x,x',x''\in X$ выполнено следующее:$$1)x \textasciitilde x(рефлексивность) 
$$$$2)x'\textasciitilde x \Rightarrow x \textasciitilde x'(симметричность)$$$$3)x'\textasciitilde x \ и \ x\textasciitilde x'' \Rightarrow \ x' \textasciitilde x''(транзитивность)$$
## Класс эквивалентности 
Классом эквивалентности,содержащим х, называется множетсво $$\bar{x}:=\{x'\in X|x'\textasciitilde x\}$$
### Теорема.Множество [[Отношения#Класс эквивалентности|классов эквивалентности]] по отношению к $\textasciitilde$ задаёт [[Разбиение|разбиение]] множества X,то есть X это объединение непересекающихся множетсв
Это почти очевидно,но всё же надо доказать.Пусть мы разбили множетсво X на сколько-то классов эквивалентности $\pi_n$ и пусть какие-то два из них пересекаются,то есть $\pi_1 \cap \pi_2=x$ ,тогда по опредлению [[Отношения#Класс эквивалентности|класса экввалентности]] получим,что $x \textasciitilde x'$  и $x \textasciitilde x''$ ,но уже по опредлению [[Отношения#Отношение эквивалентости|отношения экивалентости]] и свойства 3 получим,что $x' \textasciitilde x''$,то есть $\pi_1=\pi_2$ ,а значит,что X разбито на непересекаэщиеся классы эевивалентности.
