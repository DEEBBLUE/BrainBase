# Определения
Оба типа размещений служат для нахождения уникальных выброк,или $упорядоченных$ выборок,вопрос лишь в том 
## Размещения без повторений
Обозначет количество способов выбрать k элементов из n доступных без повторений,то есть каждый элемент выбирается только 1 раз записывается и выражается следующим образом:$$A_{n}^k=\frac{k!}{(n-k)!}$$
## Размещения с повторением
Обозначает колчество способов выбрать k элементов из n доступных,при учёте ,что уже выбранные элементы не убираются из доступных значений.Обозначается как $$\bar{A^{k}_{n}}=n^k$$
