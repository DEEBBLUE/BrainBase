# Simple 
времена simple - это простые факты или события,которые повторяются через определённые промежутки времени

## Past Simple 
Простое прошедшее время образуется следующим образом: $$Подл. \ \ + \ \ V_{-ed}/V_2$$ Если же глагола нет,тогда нужно использовать глагол to be во 2-ой форме,то есть were 
Это время означает действие которое произошло в прошлом :
>I want to England last year 

### Слова маркеры 
На past simple указывают слова ,которые отвечают на вопрос "когда?" то есть :
#### 1.Словосочетания с "last":
last year
last night
last month 
#### 2.Cловосочетания с "ago"(тому назад):
five minutes ago 
three days ago 
long ago(давно)
#### 3.Конкретные моменты времени в прошлом
In $date$ 
On $date$
at $the \ \ time$
#### 4.Выражения с then/that
then(тогда,в тот момент)
at that time(в это время)

#### 5.Общие указатели на прошлое
Once(однажды)
Previously(ранее)
In the past

## Present Simple 
Present Simple отвечает за те события которые повторяются регулярно и за простые факты 
Образуется как:$$Подл. \ + \ V_{1}/V_{s}(для \ \ he |she|it) $$
Если нет действия,тогда используется глагол to be в соотвествующей форме am/is/are 
### Слова маркеры 
Все слова выражающие передичность действия,то есть 
often,usually,always,sometimes,every ...,at the ...,on ...
## Future Simple 
Future simple выражает действия,которые происходут в будущем времени 
Образуется ,как $$Подл. \ + \ will \ +\ V_1 $$
# Continuous(Progressive)
Обозначает действия которые происходят(происходили,будут происходить) в момент разговора,то есть действие началось до разговора,продолжается в данный момент и не вожно когда оно закончится
## Past Continuous 
Обознчает действие которое происходило в конкретный момент в прошлом.Строится следующим образом $$Подл. \ \ + \ \ was/were \ \ + \ \ V_{-ing}$$ Was пишут в случае единственного числа 
Например I was driving car at 7 a.m. o'clock yesterday(Я вёд машину вчера)
### Слова маркеры
at $time$ o'clock yesterday
all day/all night
while (в то время как)
when (когда)
## Present Continuous
Обозначает действие которое происходит сейчас.Строится как $$Подл. \ \ + \ \ am/is/are \ \ + \ \ V_{-ing} $$ am/is/are выбрается по тем же соображениям,что и в [[Времена#Simple|simple]] 
### Слова маркеры
Now
Right now
At the moment
Currently(в текущее время) 

## Future Continuous 
Обозначает действие которое будет происходить в конкретный момент в будеющем.Сторится как $$Подл. \ \ + \ \ will \ \  be \ \ + \ \ V_{-ing}$$
